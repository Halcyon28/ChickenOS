So this is a webOS thats chicken themed and has a few hidden secrets and functions. This webOS was inspired by Windows 7 Operating System.  

test it out here:  
chicken-os-kappa.vercel.app

Functionalities include:

Chicken Note — a fully functional text editor  
Cluck Clock — a live clock that functions like a normal clock  
Egg Counter — track eggs with a log (don't know why I made this)  
Chicken Facts —facts about chickens  
Egg Painter — a drawing canvas with save support  
CoopBeats — A music player with some pre-downloaded songs  
Egg Catch— A game where you need to catch eggs in a basket.  
About ChickenOS— Info about the OS that shows devlogs and functions
